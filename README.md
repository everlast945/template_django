# template_django
Шаблонный проект, где уже есть частичные настройки


Этапы:
- Переименовать проект
- Проставить версии пакетов и расскомментировать нужные
- Заполнить файл .env
```
# Минимум так
SECRET_KEY=XXXXX
DEBUG=on
SQL_DEBUG=on
ALLOWED_HOSTS=*
DATABASE_URL=psql://username:password@localhost:5432/db_name
```
